# 🚍 Transport Data Analysis Dashboard

## 📊 Project Overview
This project analyzes public transport data to understand travel patterns, passenger usage, and key trends.  
An interactive dashboard was created to visualize insights and help interpret transport data effectively.

The dashboard enables users to explore transport statistics, identify trends, and analyze passenger movement patterns.

---

## 🛠 Tools & Technologies
- Microsoft Power BI
- Data Analysis
- Data Visualization
- CSV Dataset

---

## 📂 Project Files
- `Transport Dashboard Final.pbix` – Power BI dashboard file

## 📈 Key Insights
- Passenger usage trends across different routes
- Peak travel times and busiest transport routes
- Comparative analysis of transport demand
- Visualization of transport data through charts and KPIs
